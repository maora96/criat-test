# cria-test

project for cria's selection process.

## technologies

- react
- nodejs
- mysql
- prisma
- express

## dependencies
- react
- axios
- prisma
- @prisma/client
- cors
- dotenv
- multer
- express

## start project

```bash
# clone repository
$ git clone https://github.com/maora96/criat-test.git

# install back-end dependencies
$ cd server
$ npm install

# run server
$ node index.js

# install front-end dependencies
$ cd client
$ npm install

# run app
$ npm start
```
